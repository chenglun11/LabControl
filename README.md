### Requirements
1. SpringBoot
2. MySQL
3. Vue.js+Element Plus
4. Mybatis
本项目前后端分离,且区分repo.
请务必通过 [前端仓库地址](https://github.com/chenglun11/LabControlVue) 打开


### Getting Started
1. Clone本项目并从IDEA打开
2. 使用IDEA自带的Maven管理器安装依赖
3. 将SQL导入数据库(其中测试数据自行处理)
4. 运行✅
